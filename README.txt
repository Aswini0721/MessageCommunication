Steps to Setup
1. Clone the application
git clone https://github.com/[PATH]

2. Build and run the app using maven
mvn clean install
Run as Java application --> MessageCommunicationApplication.java 

3. Excute the sql scripts documented in docs folder.

4.Hit the URL in different browser by giving name : http://localhost:7777/ or http://localhost:7777/login - To login 

5. Start communication, If any user wants to leave, click on Logout.

6.Hit the URL: http://localhost:7777/list- To show the previously recored history.

7. Visit back to Login page by clicking on Send Messages.
